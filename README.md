# プレ卒用サンプルコード
Q.これはなに？  
A.プレ卒用のOpenCVのサンプルコードです。  
  
  
Q.どれがどのファイル？  
A.「binarization」  ：二値化  
  「contrast」      ：コントラスト補正  
  「copyimage」     ：複製  
  「edge」          ：エッジ処理  
  「fliphorizontal」：左右反転  
  「gamma」         ：ガンマ補正  
  「grayscale」     ：グレースケール変換  
  「rot90」         ：90度回転

A.展開後、Sorce Filesに突っ込んでください。
